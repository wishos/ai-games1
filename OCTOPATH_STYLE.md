# 八方旅人风格 - HD-2D 像素冒险

## 参考风格
- **Octopath Traveler**: 像素角色在精致预渲染/3D背景上
- 像素角色: 16x16 或 32x32，chunky pixels
- 背景: 油画感风景，暖色调，体积光，粒子效果
- UI: 复古 JRPG 风格，优雅边框

## 视觉元素
- **像素角色** × **油画风景** 的结合
- 背景: 2560x1440 预渲染风格场景
- 角色: 清晰像素边缘 (无抗锯齿)
- 色调: 暖橙 + 深蓝 + 金色高光
- 动态光效: 阳光穿透云层、粒子飘落
- 景深: 背景模糊模拟浅景深

## 场景
- 草地: 暖绿色 + 黄色小花 + 飘落粒子
- 城镇: 砖墙 + 木质建筑 + 暖黄灯光
- 地牢: 阴暗蓝调 + 火把光效 + 雾气
- Boss房: 红色/紫色暗调 + 雷电特效
