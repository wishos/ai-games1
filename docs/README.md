# 江湖地牢 - Martial Arts Roguelike

> 武侠风格回合制 RPG，HD-2D 像素风格

## 🎮 游戏概述

一款以**武侠江湖**为背景的地牢冒险 RPG，
采用**HD-2D 像素风格**，玩家扮演初入江湖的年轻侠客，挑战8层地牢。

### 核心特色
- 🎨 **HD-2D 视觉**：程序化生成的武侠风景背景
- ⚔️ **回合制战斗**：攻击/防御/技能/道具/逃跑，8职业完整技能树
- 👥 **8大职业**：战士/法师/猎人/盗贼/牧师/骑士/吟游诗人/召唤师
- 🗺️ **8层递进地牢**：从草寇山贼到武当真人
- 💾 **存档系统**：支持3个存档槽
- 📜 **任务系统**：任务日志追踪，自动进度更新

## 🎮 操作指南

### 运行游戏
1. 安装 [Godot 4.3](https://godotengine.org)
2. 打开 `project.godot`
3. 点击 ▶️ 运行

### 操作
- **WASD**: 移动
- **撞墙**: 遇敌
- **找到楼梯**: 进入下一层

## 📝 开发规范

### 代码风格
- 使用 GDScript
- 类名大写开头：`PlayerData`, `EnemyData`
- 变量小写下划线：`hp`, `max_hp`
- 常量全大写：`const PALETTE`

### Git 提交规范
```
feat: 新功能
fix: 修复bug
docs: 文档更新
refactor: 重构
chore: 杂项
```
