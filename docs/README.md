# 八方旅人 - Octopath Adventure

> HD-2D 像素风格的 Mac 原生 RPG 游戏

## 🎮 游戏概述

一款致敬《八方旅人》(Octopath Traveler) 的 2D RPG 游戏，
采用**油画风景 + 像素角色**的 HD-2D 视觉风格。

### 核心特色
- 🎨 **HD-2D 视觉**：程序化生成的油画风格背景
- ⚔️ **回合制战斗**：攻击/防御/技能/道具/逃跑
- 👥 **8大职业**：各有独特技能树
- 🗺️ **8层递进地牢**：最终挑战远古巨龙
- 💾 **自动存档**：本地进度保存

## 🎮 操作指南

### 运行游戏
1. 安装 [Godot 4.3](https://godotengine.org)
2. 打开 `project.godot`
3. 点击 ▶️ 运行

### 操作
- **WASD**: 移动
- **撞墙**: 遇敌
- **找到楼梯**: 进入下一层

## 📝 开发规范

### 代码风格
- 使用 GDScript
- 类名大写开头：`PlayerData`, `EnemyData`
- 变量小写下划线：`hp`, `max_hp`
- 常量全大写：`const PALETTE`

### Git 提交规范
```
feat: 新功能
fix: 修复bug
docs: 文档更新
refactor: 重构
chore: 杂项
```
