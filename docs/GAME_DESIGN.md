公式：
- `HP增长` = `maxHp + 10` 每级
- `MP增长` = `maxMp + 3` 每级
- `ATK增长` = `atk + 2` 每级
- `DEF增长` = `def + 1` 每级
- `SPD增长` = `spd + 1` 每级
- `LUK增长` = `luk + 1` 每级
- `升级EXP` = `floor(level^1.5 * 60)`

### 战斗公式

- **普攻伤害** = `ATK - 敌人DEF + random(-3, 3)`，最小为1
- **暴击** = `random(100) < LUK * 2`，暴击伤害 x2
- **技能伤害** = `ATK * 1.8 - 敌人DEF + random(-5, 5)`
- **逃跑成功率** = 60%
- **防御减伤** = 50%

### 技能详解

#### 战士
- **猛击** (MP:10): ATK * 1.5 伤害
- **防御** (MP:0): 减少50%伤害，+5 MP回复
- **冲锋** (MP:20): ATK * 2.5 伤害，敌人眩晕1回合

#### 法师
- **火球** (MP:15): ATK * 2.0 伤害
- **冰霜** (MP:15): ATK * 1.8 伤害，敌人SPD-2
- **闪电** (MP:25): ATK * 3.0 伤害

#### 猎人
- **狙击** (MP:10): ATK * 2.0 伤害，必定命中
- **陷阱** (MP:15): 战斗开始造成 ATK * 1.5 伤害
- **毒箭** (MP:20): ATK * 1.5 + 每回合5伤害，持续3回合

### 敌人行为AI设计

```
AI决策优先级:
1. 血量危险 (<20%) → 使用治疗/逃跑技能
2. 有增益效果可叠加 → 继续增益
3. 目标有debuff可利用 → 执行收割
4. 普通攻击间隔 → 普攻

特殊机制:
- 群体怪物: 50%概率集火血量最低目标，50%概率随机目标
- 精英怪: 每3回合释放一次特殊技能
- Boss: 阶段转换时必有特殊动画和台词
```

## 📖 剧情故事

### 故事背景

千年前，**"大灾变"之夜**，天穹裂开，魔力如洪水般涌入人间。
各族联合英雄封印了邪龙**"风神"Zephyranthes**，代价是三位英雄的生命与大陆的和平。
千年后，封印开始松动。巫妖王 Malachar 领导的亡灵帝国从东境扩张，
矮人王国陷落，精灵圣地银月森林被黑雾侵蚀……
人类王国瓦尔德的冒险者公会发出召集令：任何能深入**龙之巢穴**、找到封印钥匙的人，将获得至高荣耀与无尽财富。
