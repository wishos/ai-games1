## [Unreleased] - v0.3

### Added
- **装备强化系统完整实现**:
  - 强化面板（商店第5个Tab）：显示武器/防具/饰品当前强化等级
  - 强化成功率：+1~+9为100%，+10开始有失败风险（+10:80%, +11:65%, +12:50%, +13:35%, +14:20%, +15:10%）

  - 强化属性加成：+1~+5每级+3%基础属性，+6~+10每级+5%，+11~+15每级+8%
  - 强化失败处理：失败不退级（保留当前强化等级）
