# CHANGELOG

## [Unreleased] - v0.2

### Added
- **浮动伤害文字系统**: 战斗中的浮动伤害数字，支持6种类型：
  - 橙黄色：普通伤害
  - 橙色（放大）：暴击伤害
  - 绿色：治疗效果
  - 蓝色：护盾获得
  - 黄绿色：中毒/DOT伤害
  - 灰色：闪避/MISS
  - 红色：减益效果
  - 淡蓝色：增益效果
- **战斗肖像动画增强**: 肖像呼吸动画、受伤闪红、治疗绿光
- 8种职业完整T1技能树（各3技能，含冷却+MP消耗）

### Improved
- Boss AI多阶段系统（远古巨龙3阶段、哥布林王狂暴等）
- 战斗UI交互体验
