# CHANGELOG

## [Unreleased] - v0.3

### Changed
- **武侠主题全面回归**: 将敌人属性表、Boss设计、故事背景全面从奇幻风格切换为武侠江湖风格
  - 替换史莱姆/骷髅/恶魔/巨龙等奇幻敌人 → 劫道山贼/血刀门弟子/江湖刺客/门派弟子
  - 替换哥布林王/巫妖领主/熔岩巨魔/堕落天使/远古巨龙Boss → 山贼王韩霸天/血刀门护法·血手赫连铁树/门派叛徒司马青云/华山掌门岳不群/武当真人张三丰
  - 故事背景重写：从"封印邪龙"改为"华山论剑后的江湖格局 + 门派秘籍失窃案"

## [Unreleased] - v0.2

### Added
- **武侠敌人分层设计**: 敌人按层数与势力分布（草寇1-2层/邪派3-4层/散人5-6层/门派7-8层）
- **完整武侠Boss体系**: 5位Boss详细设计（山贼王→血刀门护法→门派叛徒→华山掌门→武当真人），含多阶段变身机制

### Added
- **层数专属敌人系统**: 新增12种敌人，按地牢层数分级刷新：
  - 1-2层: 史莱姆、洞穴蝙蝠、野猪
  - 3-4层: 骷髅战士、哥布林、幽灵
  - 5-6层: 深渊恶魔、兽人战士、暗黑骷髅
  - 7-8层: 暗影刺客、暗黑骑士、远古巨龙
- 每种敌人均有独特像素精灵图和颜色标识
- **浮动伤害文字系统**: 战斗中的浮动伤害数字，支持6种类型：
  - 橙黄色：普通伤害
  - 橙色（放大）：暴击伤害
  - 绿色：治疗效果
  - 蓝色：护盾获得
  - 黄绿色：中毒/DOT伤害
  - 灰色：闪避/MISS
  - 红色：减益效果
  - 淡蓝色：增益效果
- **战斗肖像动画增强**: 肖像呼吸动画、受伤闪红、治疗绿光
- 8种职业完整T1技能树（各3技能，含冷却+MP消耗）
- **战士T2技能（狂战士路线）**:
  - 血之狂暴: ATK+30%，每回合自损10HP，持续3回合
  - 旋风斩: ATK×1.2×2-4次攻击
  - 战吼: 自身ATK+40%，持续2回合
- **法师T2技能（元素大师/奥术师路线）**:
  - 流星火雨: ATK×0.8立即伤害+3回合灼烧DOT
  - 霜冻领域: ATK×1.5伤害+敌人速度-50%持续2回合
  - 连锁闪电: ATK×2.5+0.3秒后ATK×1.5连锁伤害
  - 魔法盾: 消耗80%MP转为护盾值
  - 法术穿透: 下2次攻击无视敌人防御
  - 魔力回旋: 每回合吸MP+回HP，持续3回合
- **法术穿透机制**: 所有伤害技能均可触发，穿透消耗次数
- **战斗buff/debuff系统**: 新增狂暴、战吼、消失、灼烧、减速、魔盾等状态

### Improved
- Boss AI多阶段系统（远古巨龙3阶段、哥布林王狂暴等）
- 战斗UI交互体验
