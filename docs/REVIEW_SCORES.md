#### 🚨 风险1（致命）：设计文档≠产品 — 范围蔓延导致永远无法完成

- **一个人的业余开发，3个月内连技能树T2都做不完**

**概率评估：** 80%可能因范围蔓延导致项目搁浅

**应对策略：**
- 砍！砍到只剩核心循环：**职业→地牢→技能升级→Boss→结局**（约3小时内容）

- 设计文档是负债，不是资产——每个"已规划"的功能都在消耗心理成本

#### 🚨 风险2（致命）：HD-2D背景完全缺失 — 视觉卖点归零

**问题本质：** HD-2D的核心是**油画风景背景**（八方旅人），不是像素角色。像素角色中国不缺，油画风景才是壁垒。

**当前状态：**
- `parallax_bg` 层全空
- `bg_layer` 无任何内容
- 玩家看到的是：纯色底 → 静态像素精灵 → 感觉像1990年代的早期GameMaker游戏

**概率评估：** 90% — 不补背景则视觉无任何竞争力

**应对策略（按成本排序）：**
1. **方案A（最快，0成本）：** 用Godot程序化生成 — 渐变天空+多边形山脉剪影+程序草地纹理，1周内可完成"勉强能看"的背景层
2. **方案B（中等成本）：** 用Stable Diffusion / Midjourney生成油画风格背景图，1920×1080，约需要20-30张，覆盖主要场景类型，成本约¥200-500（MiniMax API）
3. **方案C（长期）：** 找外包画师制作，¥2000-5000可完成一套可用的HD-2D背景组

### 🚨 TOP 3 商业风险

#### 1.【致命】范围蔓延 — 300KB设计文档是陷阱

300KB的设计文档是一把双刃剑：它展示了雄心，但也制造了"功能债务"。开发者（尤其是AI自动迭代时）会不断被文档中的"待做内容"吸引，而非聚焦核心循环。

**后果：** 6个月后回头看，代码可能只增加了10%，但文档又扩大了50%。

**解法：** 把 `GAME_DESIGN.md` 中 v0.2-v0.6 的内容**全部冻结**，v1.0只做"核心循环"。新建 `ROADMAP_V1.md`，只写接下来2周要做的事。

#### 2.【高】美术资产缺口 — HD-2D背景不补完则全盘皆输

像素角色是基础分，背景才是HD-2D的附加价值。没有背景，这个游戏和"随便一个RPG Maker游戏"没有视觉差距。

**后果：** 截图无法吸引玩家，商店页面无竞争力，转化率<1%。

**解法：** 立即启动背景资产计划（程序化或AI生成），不等待其他功能完成。

#### 3.【高】口碑风险 — 未完成的Early Access可能砸招牌

Steam的评价是长期资产，一个差评轰炸的项目很难翻身。¥10以下的低价游戏玩家期待值两极化：要么"便宜有好货"的惊喜，要么"浪费10块钱"的愤怒。

**后果：** 早期100个评价若平均低于60%，算法会彻底埋葬这个游戏。

**解法：** 要么做到"¥10物有所值"再上架，要么定¥3-5低价冲量+口碑，慢慢提价。

### ✅ 优点

**1. 框架结构完整，叙事节奏尚可**
- 序章 + 6章主线 + 终章，多结局（A英雄/B龙裔/C牺牲）设计有层次感
- 每章有明确目标、奖励、剧情影响，结构意识好

**2. NPC行为设计有亮点**
- 铁匠老郑/李婶/孙药郎/鬼市商人的深度行为树（踢馆事件/深夜来客/诊脉系统/砍价小游戏）展现了对NPC立体化的追求
- 艾拉的"牧师秘密"支线（神圣血统）、雷昂的"骑士誓言"支线（父亲是封印风神三英雄之一）有人物挖掘潜力
- 希尔薇娅的隐藏身份（精灵女王私生女）是加分项

**3. 江湖传闻系统增加氛围感**
- 每日江湖传闻（任务线索/动态/隐藏/商机/警告/趣闻）设计精细，是武侠/江湖氛围的好载体

**4. 多结局加分**
- 龙裔结局（主角被龙血改变，非人非龙）概念有趣，不是简单的好/坏结局
- 牺牲结局有情感冲击力

### ❌ 缺点

#### 🔴 【致命】武侠与西式奇幻严重割裂——两套世界观强行拼接

**这是最大问题，决定了3.5/10的核心依据。**

文档标题明确是**"武侠像素RPG"**，武侠内容规划极为丰富：
- 门派体系：少林/武当/华山/峨眉
- 邪派势力：血刀门/日月神教（东方不败教主）
- 江湖散人：刺客/赏金猎人
- 江湖传说Boss：独孤求败/傅红雪/东方不败/洪七公/岳不群
- 武侠技能体系：刚柔/剑系/刀系/掌系/暗器系/毒系
- 武侠场景：门派建筑/江湖集市/悦来客栈/醉仙楼

**但"📖剧情故事"章节（第6章起）主线却是完全西式奇幻：**
- 瓦尔德王国（蓝底金狮国旗）
- 冒险者公会
- 亡灵帝国/巫妖王马查尔
- 堕落天使 Seraphiel
- 风神龙 Zephyranthes

**两套体系并存几乎没有交集。** 玩家开局进入"冒险者公会"注册成为"冒险者"，屠的是"风神龙"而不是某个武林公敌。武侠元素降格为敌人类型和场景装饰，**不是故事核心**。

此外还存在混搭问题：
- 艾拉是"随队牧师"（典型西式奇幻）而非江湖郎中
- "铁匠老郑"和"牧师艾拉"并存于同一世界观
- 门派掌门做Boss，但门派本身在主线的意义为零

#### 🟡 【严重】故事核心是经典屠龙叙事，吸引力中等

"千年前大灾变→千年后封印松动→冒险者公会召集→屠龙"是**非常成熟的叙事框架**，优点是节奏稳定，缺点是没有独特性——独立游戏市场上这类故事一抓一大把。

武侠题材被降格后，故事失去了**"江湖恩怨+门派纷争+武学境界"**这个最能体现武侠差异化的核心卖点。

#### 🟡 【严重】NPC对话极少，人物靠背景文字撑场

- **老郑/李婶/孙药郎有详细行为设计但几乎无故事对话**——他们存在于副本/支线系统，不存在于主线故事
- **主线NPC艾拉/雷昂/希尔薇娅有背景但对话量极少**——没有"关键时刻台词"，人物靠背景叙述而非行动展现
- 章节间的故事推进大量依赖**叙述者文字**而非角色互动

**具体例子：** 序章中公会管理员玛格丽特的对话只有三行，没有表情描写，没有性格展示。艾拉/雷昂作为随队NPC，他们的台词在当前设计中几乎没有。

#### 🟡 【中等】支线任务与主线脱节

- 支线任务（失踪村民/牧师秘密/黑街之王/矮人遗愿/世界树低语）设计丰富，但**完全独立于主线故事章节**
- 没有任何支线任务在章节间穿插作为"剧情补充"
- 铁匠老郑的踢馆事件可以成为很好的武侠情节，但未与主线挂钩

### 🔧 改进建议

#### 1. 【根本性】重构故事核心——真正做武侠叙事

**选择A（推荐）：武侠为本，西式奇幻元素辅助**
